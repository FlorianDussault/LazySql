## Installation

Use NuGet ([LazySqlStandard.Engine](https://www.nuget.org/packages/LazySqlStandard.Engine/)) !

Packet manager:
```sh
PM> NuGet\Install-Package LazySqlStandard.Engine -Version 1.0.10-alpha
```

.NET CLI:
```sh
> dotnet add package LazySqlStandard.Engine --version 1.0.10-alpha
```

Paket CLI:
```sh
> paket add LazySqlStandard.Engine --version 1.0.10-alpha
```

Paket CLI:
```xml
<PackageReference Include="LazySqlStandard.Engine" Version="1.0.9-alpha" />
```
